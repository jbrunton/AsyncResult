bind

fun <S, T> AsyncResult<T>.bind(onSuccess: (Success<T>) -> S, onLoading: (Loading<T>) -> S, onFailure: (Failure<T>) -> S): S

Returns the result of onSuccess if the result is AsyncResult.Success, the result of onLoading if the result is AsyncResult.Loading, and the result of onFailure if the result is AsyncResult.Failure.
