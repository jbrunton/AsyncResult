getOrNull

fun <T> AsyncResult<T>.getOrNull(): T?

Returns the value of the result if it exists and null otherwise.

  • For AsyncResult.Success this will be value.
  • For AsyncResult.Loading this will be cachedValue if it exists and null otherwise.
  • For AsyncResult.Failure this will be cachedValue if it exists and null otherwise.
