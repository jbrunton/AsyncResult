getOr

fun <T> AsyncResult<T>.getOr(defaultValue: T): T

Returns the value of the result if it has one and the given defaultValue otherwise.

  • For AsyncResult.Success this will be value.
  • For AsyncResult.Loading this will be cachedValue if it exists and defaultValue otherwise.
  • For AsyncResult.Failure this will be cachedValue if it exists and defaultValue otherwise.
