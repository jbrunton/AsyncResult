get

fun <T> AsyncResult<T>.get(): T

Returns the value of the result.

  • For AsyncResult.Success this will be value.
  • For AsyncResult.Loading this will be cachedValue if it is not null. Otherwise a NullPointerException will be thrown.
  • For AsyncResult.Failure this will be cachedValue if it is not null. Otherwise the error will be thrown.
